# 🧮 Calculator in Python Language

A simple yet effective calculator built using **Python**.  
This project demonstrates fundamental programming concepts such as functions, conditional statements, and user input handling.

---

## 📌 Features
- Basic Mathematics
- Algebra
- Matrix Operations
- Geometry
- Unit Converter
- Number System
- Statistics
- Trignometry
- User-friendly input prompts
- Error handling for invalid inputs
- Clean and modular code structure
